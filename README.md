# 🎥 Motion-Based Mouse Mover

Control your computer mouse using hand motion — no physical mouse required.

This project uses **Computer Vision + Automation** to enable touchless interaction
with your system using a webcam.

---

## 🚀 Features

- 🖐️ Hand motion–based cursor movement
- 🖱️ Gesture-based left & right click
- 🎥 Live webcam feed
- 📊 Interactive dashboard with charts
- 📹 Video recording of motion session
- 📸 Screenshot capture
- 🌙 Modern responsive UI (Bootstrap)
- ⚡ Smooth & stable cursor movement

---

## 🧠 How It Works

1. Webcam captures real-time frames
2. OpenCV detects motion regions
3. Motion mapped to screen coordinates
4. Cursor moves using PyAutoGUI
5. Gestures trigger mouse clicks
